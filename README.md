# AI/ML Internship Tasks

## Task Objective

The objective of this task is to perform Exploratory Data Analysis (EDA) and basic machine learning on a dataset. This includes understanding the data structure, identifying patterns, and visualizing relationships between features.


## Dataset Used

* **Dataset Name:** Iris Dataset
* **Source:** UCI Machine Learning Repository / Kaggle
* **Description:**
  The dataset contains measurements of iris flowers:

  * Sepal Length
  * Sepal Width
  * Petal Length
  * Petal Width
  * Species (Target Variable)


## Models Applied

The following techniques and models were used:

* Data Preprocessing
* Exploratory Data Analysis (EDA)
* Visualization using Matplotlib and Seaborn
* (Optional if applied):

  * Logistic Regression
  * K-Nearest Neighbors (KNN)
  * Decision Tree


## Key Results & Findings

* Clear separation observed between species using petal length and petal width
* Setosa class is linearly separable from others
* Strong correlation between petal length and petal width
* Minimal overlap between Versicolor and Virginica classes


## Visualizations

* Scatter plots showing feature relationships
* Histograms showing feature distributions
* Box plots identifying outliers


## Tools & Libraries Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Scikit-learn


## Conclusion

The Iris dataset provides a simple yet powerful way to understand classification problems. Visualization helps in identifying patterns, and basic ML models can achieve high accuracy on this dataset.
